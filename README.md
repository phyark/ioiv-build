这是一个便于个人快速开发 npm 包开发模板项目，基于 `Rollup` 构建，支持 `ES6` 语法， `ES5` 兼容模式，支持 `ESLint` 代码规范检查，支持 `Jest` 单元测试，支持 `Coveralls` 覆盖率报告。
